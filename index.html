How to play Raft with mods?

– Download latest Raft-ModKit release
– Unpack .zip archive whereever you want
– Put your mods inside /plugins folder (to update the list of plugins ingame press «reload all scripts»)
– Start Launcher.exe and enjoy playing modded Raft! Press M to open ModKit menu

How to create own mods?

– Create a C# .Net Framework Library project Modname_Script
Please note that assembly filename should look like ModnameScript.dll
– Add references to UnityEngine and Assembly-CSharp.dll
– Extend your script class with UnityEngine.MonoBehaviour
– Now you can accept Unity3D scene messages! More info

What are the «sample scripts»?

NoSharkScript – disables sharks
CheatScript – enables Raft's builtin cheat mode (uses F- buttons)
WeatherScript – allows to change ingame's weather (press N to open script's menu)
